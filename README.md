# AudioVisualizer
An Audio Visualizer using python and arduino

## Hardware Used:
1. Arduino Uno x1
2. WS2812B LED Strips (Made into a LED Panel) 5x5
3. Jumper wires

## Wiring
- Arduino 5V -> WS2812B 5V
- Arduino GND -> WS2812B GND
- Arduino D6 -> WS2812B DataPin

## Libraries
### Python
- numpy
- sounddevice
- serial

### Arduino
- FastLED
